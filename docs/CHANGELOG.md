# Changelog

All notable changes to the Deep Link RPA Field Inventory are documented here.

Format: [Semantic Versioning](https://semver.org/) (MAJOR.MINOR.PATCH)
- **MAJOR**: Sections added/removed, structural reorganization
- **MINOR**: Fields added/removed within existing sections
- **PATCH**: Typos, clarifications, option value changes

---

## [v1.0.0] - 2026-02-18

### Initial Release
- 19 sections covering all deep link offer form fields
- Based on screenshots taken 2026-02-17
- Non-RPA fields marked with `+` annotation
- Calculated/convenience fields marked with `(calculated)`

### Non-RPA Fields (VoicePilot additions)
- Seller Name / Email / Phone (workflow convenience)
- Escalation Clause section (additional terms functionality)
- Expanded BRBC (retainer, exclusivity, dates, notes)
- Termite Inspection Paid By
- Percentage variants for deposit, credit, down payment (calculated)

### Removed from earlier draft
- Buyer Email, Phone, Address (not on form)
- "Married" entity type (not on form)
